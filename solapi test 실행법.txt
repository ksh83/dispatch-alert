이 단계대로 하면 SMS를 쉽게 발송할 수 있습니다.Solapi SMS API를 이용해 실제 문자 보내는 실행 방법은 아래와 같습니다:

1. Solapi 회원가입 및 인증
Solapi 공식 홈페이지(https://solapi.com)에서 회원가입 후 로그인합니다.

개발자 센터에서 API Key와 Secret을 발급받습니다.

'발신번호 등록' 메뉴에서 사용할 번호를 등록하고 인증합니다.

2. Python 패키지 설치
터미널에서 아래 명령어 입력:

text
pip install solapi
3. 코드 파일 작성
아래 코드를 복사해 send_sms.py로 저장합니다. API 키, Secret, 발신번호, 수신번호를 자신의 정보로 입력하세요.

python
from solapi import SolapiMessageService
from solapi.model import RequestMessage

api_key = "YOUR_API_KEY"
api_secret = "YOUR_API_SECRET"

message_service = SolapiMessageService(
    api_key=api_key,
    api_secret=api_secret
)

message = RequestMessage(
    from_="발신번호",        # 인증한 발신번호
    to="수신번호",          # 수신인 번호
    text="안녕하세요! Solapi SMS 발송 테스트입니다."
)

try:
    response = message_service.send(message)
    print("메시지 발송 성공!")
    print(response)
except Exception as e:
    print(f"메시지 발송 실패: {str(e)}")
4. 코드 실행
코드 파일이 있는 폴더에서 아래 명령어로 실행합니다:

text
python send_sms.py
메시지 발송 성공/실패 여부가 출력됩니다. 오류가 있으면 발신번호/키/잔액/설치 여부를 점검하세요.

이 과정대로 하면 SMS 문자 송신이 완료됩니다. 더 자세한 내용과 추가 기능은 Solapi 공식 문서를 참고할 수 있습니다.
